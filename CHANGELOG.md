## [0.1.0] - 2026-03-16
### Added
- Core PowerShell script for automatic photo organisation
- EXIF date reading for JPG, HEIC, PNG, GIF
- Support for MP4, MOV, MPG, VOB video files
- Duplicate detection using MD5 hashing
- Hidden file support
- Automatic Year and Month folder sorting
- Progress tracking and log output
