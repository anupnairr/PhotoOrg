# Changelog

## [1.0.0] - 2026-03-20
### Added
- Automatic photo and video organiser
- Reads EXIF date for accurate year and month sorting
- Supports JPG, HEIC, PNG, GIF, MP4, MOV, MPG, VOB and more
- Duplicate detection using MD5 hashing
- Duplicates moved safely to review folder
- Clean progress bar and activity log
- Works on Windows 10 and Windows 11
